Markdown Syntax

Headers

Use the # symbol to create headers:

# H1
## H2
### H3

Emphasis

Use * or _ for italic text, and ** or __ for bold text:

*italic*
_italic_
**bold**
__bold__

Lists

Create unordered lists with *, +, or -, and ordered lists with numbers:

* Item 1
* Item 2
1. First
2. Second

Links

Create links using the following syntax:

[title](https://www.example.com)

Images

Embed images using a similar syntax:

![alt text](image.jpg)

Blockquotes

Create blockquotes using the > symbol:

> This is a blockquote

Code

Inline code can be created with backticks `, and blocks of code with triple backticks ```:

`inline code`
```
block of code
```

Horizontal Rules

Create horizontal rules using three or more asterisks, dashes, or underscores:

***
---
___
